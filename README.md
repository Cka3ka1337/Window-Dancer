## Quick start
1. Copy repository:

    ```git
    git clone https://github.com/Cka3ka1337/Window-Dancer.git
    ```
2. Installing dependencies:
  
    ```shell
    cd Window-Dancer
    pip install -r requirements.txt
    ```
